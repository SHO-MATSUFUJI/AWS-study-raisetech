## Lightsailとは

Amazon Lightsailとは設定が簡単で料金が定額のオールインワン型クラウドサーバーサービスです。OSやアプリがあらかじめ用意されたブループリントが用意されており、WebサイトやWebアプリを素早く立ち上げたいときに向いていて、AWS知識が浅い人や、小規模運用時、すぐに動かしたい人に利用が推奨される。

## WordPressとは

Webサイトやブログを作成・管理するためのCMS(コンテンツ・マネジメント・システム)です。主な特徴として、**無料のオープンソースである**、**専門的なコーディング知識が不要**、**デザインの自由度が高い**、**機能追加が簡単**、**SEO（検索エンジン対策）に強い**などがあり、世界中の多くのサイトで採用されています。2024年の調査記事によると世界のWebサイトの約43%前後がWordPressで構築されているという紹介があります。

## EC2とLightsailでの環境構築の違い

- Lightsail - ブループリントがあり設定が簡単で素早く、知識が少なくても環境構築できる。手間が省けるが自由度は制限される。料金体制が定額である。
- EC2 - 必要なリソースを個別にインストール及び設定するため、専門知識が必要である。手間はかかるが構成を自由にカスタマイズできる。料金は従量課金制である。
